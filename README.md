**Project Name**
Employee Payroll Management System

**Description**
The Employee Payroll Management System is a Java and MySQL based application
developed to automate payroll processing in an organization.

**Technologies Used**
- Java
- MySQL
- JDBC
- Eclipse 

 **Features**
- Employee record management
- Attendance tracking
- Payroll calculation
- Salary generation
- Tax and deduction handling

**How to Run the Project**
1. Open the project in Eclipse or IntelliJ
2. Configure MySQL database
3. Run Main.java file
   
**Project Type**
CLI
